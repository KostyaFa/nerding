реализовать простое приложение с хранением и обновлением данных о пользователе

1. GET /user/{id}
страница будет возвращать юзера по id
2. POST /user/{id}
добавление пользователя
3. PUT /user/{id}
обновление данных о пользователе
4. DELETE /user/{id}
удаление данных о пользователе

для п.2 реализовать сохранение utm-меток (это всё, что идет в URI после знака вопроса)
